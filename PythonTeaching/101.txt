1.字符串(str)类型
 - 用单引号或双引号 指定一个 字符串
 - 用三引号 指定一个 多行字符串
 - 单引号里面可以用双引号 反之同理
 - 转义符 "\" 用 \n 换行
 - 用加法来连接字符串 用乘法来重复字符串
2.整数(int)类型
3.小数(float)类型
4.print() 函数
 - 括号里放表达式，可以是数字、计算、字符串等
5.*某类型*() 函数
 - 小数转整数 舍去小数部分
6.定义一个函数
 - 格式：
    def name(something1: type, ...) -> tips:
        return something...
7.while循环
 - 格式：
    while condition:
        # Do something...
8.变量赋值
 - 格式：
    name = value
9.逻辑
 - 是否相等 = =
 - 大于 小于 大于等于 小于等于 略
 - 不等于 ! =
 - 逻辑与或非 and or not
 - 括号优先
10.for循环
 - for可以从很多内容中逐个取出赋给新的变量
 - 格式：
    for name in manyThings:
        # Do something...
